# 💬 Chat Application

A Python-based Chat Application that enables real-time communication between users using a client-server architecture. This project was developed as part of the **Python Programming Internship at Oasis Infobyte (OIBSIP)**.

## 📌 Project Overview

The Chat Application allows users to exchange messages in real time through a network connection. It demonstrates the implementation of socket programming, networking concepts, and client-server communication using Python.

## ✨ Features

* Real-time messaging
* Client-server architecture
* Multiple user communication support
* Fast message transmission
* Simple and easy-to-use interface
* Reliable communication over a network

## 🛠️ Technologies Used

* Python 3
* Socket Programming
* Networking Concepts
* Threading (if used)

## 🚀 How to Run

### Step 1: Clone the Repository

```bash
git clone https://github.com/YourUsername/OIBSIP.git
```

### Step 2: Navigate to the Project Folder

```bash
cd OIBSIP/Chat_Application
```

### Step 3: Start the Server

```bash
python server.py
```

### Step 4: Start the Client

Open a new terminal and run:

```bash
python client.py
```

### Step 5: Begin Chatting

Enter messages from the client side and communicate in real time.
