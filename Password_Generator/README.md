# 🔐 Random Password Generator

A Python-based Random Password Generator that creates strong and secure passwords based on user preferences. This project was developed as part of the **Python Programming Internship at Oasis Infobyte (OIBSIP)**.

## 📌 Project Overview

The Random Password Generator helps users create secure passwords by allowing them to specify password length and choose different character types such as letters, numbers, and special symbols. The generated passwords are randomized to enhance security and protect user accounts.

## ✨ Features

* Generate secure and random passwords
* User-defined password length
* Include uppercase and lowercase letters
* Include numbers and special characters
* Easy-to-use interface
* Fast and efficient password generation

## 🛠️ Technologies Used

* Python 3
* Random Module
* String Module
